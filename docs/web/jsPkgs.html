Dojo 1.1

Dojo 1.1

This Javascript tool kit is used to support the new browser interface. A tarred and gzipped Version 1.1 from the Dojo download site is distributed with OSCARS. You can run the Ant task "ant installdojo" to install it, or let it automatically install when you deploy the war file with Ant. If you were running the server with the older interface before, be sure to stop the server, completely remove webapps/OSCARS before deploying the war file with "ant deploywbui", and restart.
