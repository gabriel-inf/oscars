Dojo 1.1

Dojo 1.1

This tool kit is used to support the new browser interface. Go to the Dojo site and download version 1.1. Untar as root into /usr/local/src. Make a symbolic link named dojo in that directory to dojo-release-1.1.0. The ant tasks depend on that link. ant deploywbui will copy all 1600 plus files to web/lib and create the OSCARS webapp. If you were running the server with the older interface before, be sure to stop the server, completely remove webapps/OSCARS before running deploywbui, and restart.
