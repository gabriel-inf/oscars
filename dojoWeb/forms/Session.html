Navigation tab functionality is as follows:

  • Clicking on the Reservations tab will list all current reservations.
  • Choosing a particular reservation from that list will bring up that reservation's details under the ReservationDetails tab.
  • Clicking on the Create Reservation tab will bring up a form to create a reservation.
  • Clicking on the Users tab brings up a list of all current users.
  • Choosing a particular user from that list brings up the User Details tab, allowing modification of a user's profile.

To log out, return to this tab, and click on the corresponding button.
